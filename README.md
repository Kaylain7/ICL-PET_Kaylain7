Personal Expenses Tracker - Lab 3
Overview

A simple command-line app to track daily expenses, manage balance, and generate reports. Includes a shell script for archiving and searching expense files.

Files

expenses-tracker.py : Python CLI app

archive_expenses.sh : Archive/search expenses

balance.txt : Starting balance

README.md : Instructions

Features

Main Menu: Check balance, view expenses, add expense, exit

Add Expense: Enter date, item, and amount; validates input; updates balance

View Expenses: Search by item name or amount

Shell Script: Archive expense files and search by date
